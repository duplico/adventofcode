# Advent of Code - Python Solution

## Setup

```bash
uv sync
```

## Running

```bash
# Part 1
uv run advent 1 input.txt
uv run advent 1 -v input.txt      # verbose
uv run advent 1 -vv input.txt     # very verbose

# Part 2
uv run advent 2 input.txt
```

## Development

```bash
# Run tests
uv run pytest

# Or directly without installing
uv run python -m advent 1 input.txt
```
