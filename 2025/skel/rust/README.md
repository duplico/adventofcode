# Advent of Code - Rust Solution

## Running

```bash
# Debug build (fast compile, slow run)
cargo run -- 1 ../input.txt

# Release build (slow compile, fast run)
cargo run --release -- 1 ../input.txt

# With verbose flag
cargo run --release -- 1 ../input.txt -v

# With sample input
cargo run -- 1 ../sample_input.txt
```

## Quick iteration

```bash
cargo watch -x "run -- 1 ../sample_input.txt"
```
