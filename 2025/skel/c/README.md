# Advent of Code - C Solution

## Build

```bash
make          # Release build
make debug    # Debug build with sanitizers
```

## Running

```bash
# Direct
./advent 1 ../input.txt
./advent 2 ../input.txt -v

# Via make
make run1
make run2
make run1-sample
make run2-sample
```

## Clean

```bash
make clean
```
